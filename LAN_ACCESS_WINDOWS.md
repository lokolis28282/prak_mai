# Доступ к ODE из локальной сети

## Первичная настройка

1. Один раз запустите `setup_lan_access_windows.bat` от имени администратора.
2. Для сетевой работы запускайте `start_lan_windows.bat`.
3. Окно ODE должно оставаться открытым. Для остановки нажмите `Ctrl+C`.

Launcher выводит доступные адреса вида:

```text
http://192.168.1.6:8765
```

Товарищ должен открыть подходящий адрес в браузере. Оба компьютера должны быть
в одной локальной сети или в одной частной VPN-сети.

Правило брандмауэра разрешает входящие подключения только на TCP-порт `8765`,
только для Python, запускающего ODE, и только из локальных подсетей.

Не настраивайте проброс порта `8765` на интернет-роутере. Текущая версия ODE
работает по HTTP без TLS и предназначена для доверенной локальной сети.
