# Скриншоты для документации

Снимки следует делать в браузерном интерфейсе ODE на демонстрационной базе. Команда ниже удаляет выбранную базу, поэтому не запускайте ее для рабочих данных:

```bash
python3 app.py --db data/demo.db seed --reset
python3 app.py gui --db data/demo.db --port 8876
```

Рекомендуемый набор:

1. `01_home.png` — главная страница и основные действия.
2. `02_balance.png` — баланс, поиск и фильтры.
3. `03_receipt_scanner.png` — список S/N перед подтверждением прихода.
4. `04_issue_scanner.png` — найденные и проблемные S/N расхода.
5. `05_deliveries.png` — реестр поставок и их статусы.
6. `06_delivery_card.png` — строки поставки и приемка сканером.
7. `07_reports.png` — отчет за выбранный период.
8. `08_administration.png` — backup, проверка базы и журнал действий.

На изображениях не должно быть реальных ФИО, паролей, документов, серийных номеров или внутренних адресов организации. Используйте отдельную демонстрационную базу и обрезайте персональные данные из адресной строки и путей.
