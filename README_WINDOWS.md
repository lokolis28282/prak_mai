# Запуск ODE на Windows

## 1. Распаковка

Распакуйте актуальный ZIP-пакет ODE в отдельную папку, например `C:\ODE`. Не запускайте программу прямо из архива.

## 2. Запуск

Установите Python 3.10 или новее с сайта python.org и отметьте пункт `Add Python to PATH`. Затем дважды щелкните `start_windows.bat`. Альтернативный запуск из командной строки:

```bat
python app.py
```

## 3. Вход

Браузер откроется автоматически. Если этого не произошло, откройте `http://127.0.0.1:8765`. Используйте существующую учетную запись: перенос базы сохраняет пользователей и пароли.

## 4. Где находится база

Рабочая база: `data\warehouse.db`. Не редактируйте и не заменяйте этот файл при запущенной программе.

## 5. Резервная копия перед загрузкой данных

Откройте `Администрирование → Резервные копии` и нажмите `Создать резервную копию`. Дождитесь сообщения об успешном завершении. Копии находятся в `data\backups`.

## 6. Загрузка прихода

Откройте `Склад → Приход`. Для сканирования заполните общие поля партии, установите курсор в поле сканера и считывайте S/N; после проверки списка нажмите `Принять всё на склад`. Сканер должен работать как клавиатура и отправлять Enter после кода.

Для CSV скачайте шаблон, выберите заполненный файл, проверьте ошибки и нажмите `Подтвердить загрузку`.

## 7. Загрузка расхода

Для сканирования откройте `Расход`, заполните задачу и общие поля, затем считывайте S/N. Найденные позиции будут показаны до подтверждения; неизвестные S/N после подтверждения попадут в проблемные строки.

Также можно найти одну позицию и нажать `Списать` либо загрузить CSV массового списания.

## 8. Проверка баланса

Откройте `Склад → Баланс`. Используйте общий поиск и фильтры. Для полной выборки нажмите `Скачать баланс`.

## 9. Поставки

В разделе `Склад → Поставки` загрузите CSV снабжения и проверьте preview: отдельно показаны новые, уже существующие, повторные и ошибочные строки, а также неизвестные столбцы. После подтверждения новые S/N принимаются на склад, а у существующих позиций заполняются только пустые реквизиты. Перед импортом сделайте резервную копию.

## 10. Если CSV открылся одной колонкой

Новые шаблоны и отчеты используют `;` и UTF-8 BOM. Если Excel все равно показал одну колонку, откройте файл через `Данные → Из текста/CSV`, выберите кодировку UTF-8 и разделитель `;`.

## 11. Завершение работы

Закройте вкладку браузера, затем вернитесь в окно ODE и нажмите `Ctrl+C`. После сообщения `Работа завершена` окно можно закрыть.
